62 Documentation
63 -------------
64 Un fichier ```Doxyfile``` est déjà fourni. Pour générer la documentation :
65 ```bash
66 $ doxygen
67 ```
68 La documentation sera générée dans un dossier ```documentation```.
69 
70 Branches et tags
71 ----------------
72 La branche ```master``` pointe sur la dernière version stable, tandis que la branche ```dev``` pointe sur la version de développement. Pour passer d'une branche à une autre :
73 ```bash
74 $ git checkout <branche>
75 ```
76 
77 Les versions majeures également sont affublées de tags. Pour lister les tags :
78 ```bash
79 $ git tag --list
80 ```
81 Ensuite, pour passer à un tag particulier :
82 ```bash
83 $ git checkout <tag>
84 ```

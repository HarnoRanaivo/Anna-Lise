static void* q_malloc_n ( size_t  n,
size_t  size 
)
inlinestatic

Allouer une zone mémoire de taille n * size, et quitter en cas d'erreur.

Paramètres
nNombre d'éléments.
sizeTaille de la zone mémoire souhaitée.
Valeurs retournées
pointeurvers la zone mémoire allouée.
NULLsi size == 0 ou n == 0.

En cas d'erreur d'allocation, cette fonction cause la fin du programme. Comme le fait malloc, si n ou size sont égaux à 0, cette fonction retourne NULL.
